# CivilX Revit Plugins - Скомпилированные версии

Дата компиляции: 19.09.2025 14:59

## Содержимое архива

### 1. DataViewer Plugin (DataHub)
- **Файл**: `RevitExporterAddin.dll` (191 KB)
- **Конфигурация**: `RevitExporterAddin.addin`
- **Описание**: Плагин для экспорта данных из Revit
- **Группа**: DataHub
- **Статус**: Требует подписки

### 2. Authorization Plugin (Настройки)
- **Файл**: `CivilXAuthPlugin.dll` (58 KB)
- **Конфигурация**: `CivilXAuthPlugin.addin`
- **Описание**: Плагин авторизации и управления настройками
- **Группа**: Настройки
- **Статус**: Всегда активен (бесплатный)

## Установка плагинов

### Требования
- Autodesk Revit 2023
- .NET Framework 4.8
- Windows x64

### Инструкция по установке

1. **Скопируйте файлы в папку Revit Addins:**
   ```
   C:\Users\[USERNAME]\AppData\Roaming\Autodesk\Revit\Addins\2023\
   ```

2. **Для DataViewer:**
   - Скопируйте `DataViewer/RevitExporterAddin.dll`
   - Скопируйте `DataViewer/RevitExporterAddin.addin`

3. **Для Authorization:**
   - Скопируйте `Authorization/CivilXAuthPlugin.dll`
   - Скопируйте `Authorization/CivilXAuthPlugin.addin`

4. **Перезапустите Revit**

## Структура плагинов

### DataViewer (DataHub)
- Экспорт данных из Revit проектов
- Управление проектами и экспортами
- Интеграция с веб-сервисом CivilX
- Требует активной подписки DataHub

### Authorization (Настройки)
- Авторизация пользователей
- Управление профилем
- Управление подписками
- Всегда активен для всех пользователей

## Поддержка

При возникновении проблем:
1. Проверьте версию Revit (должна быть 2023)
2. Убедитесь, что .NET Framework 4.8 установлен
3. Проверьте права доступа к папке Addins
4. Перезапустите Revit после установки

## Обновления

Эти плагины соответствуют актуальной версии сайта CivilX с реорганизованной структурой плагинов:
- Группа "Data" переименована в "DataHub"
- Создана новая группа "Настройки" для плагина авторизации
- Плагин авторизации всегда активен и не требует покупки
